## Sneak Attack
*By unknown*

This snippet is added to the end of an attack line

**Alias:**  
*Dicecloud/Beyond:*  
```GN
!snippet sneak -d1 "{{ceil(RogueLevel/2)}}d6 [piercing]"
```
*Other sheets:*  
```GN
!snippet sneak -d1 "{{ceil(level/2)}}d6 [piercing]"
```
**Use With:**  
``!i a [monster] [weapon] sneak`` or ``!a [weapon] sneak``
